Installing gwcosmology
---------------------

The easiest method to install gravityspy is using `pip <https://pip.pypa.io/en/stable/>`_ directly from the `GitHub repository <https://github.com/gwcosmology/gwcosmology.git>`_:

gwcosmology software has been tested on py35 and py27. Work is in progress to have unit tests verify gravity spy on py27 and py34 py35 and py36. 

.. code-block:: bash

   $ pip install git+https://github.com/gwcosmology/gwcosmology.git

For more details see :ref:`install`.
